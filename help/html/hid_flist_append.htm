Appending New Entries to a File List

on the file list dialog

Append tells Moderate that you want to add one or more new file list entries to the end of your file list. (You can also insert new entries at the current cursor position in the file list - see Inserting New Entries into a File List.)

When you click on the Append button, this menu pops up. Click on the various menu options for more information on each insertion option.
