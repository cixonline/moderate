Edit File List command

File Directory tab

Moderate provides powerful tools for maintaining CIX file lists. These are centred on the main file list editing dialog that allows maintenance of both the file list and the CIX file directory underlying it.

Read Maintaining File Lists for information on how file lists are organised.

The file list editor dialog has two tabs, one for editing the moderator's file list itself and one for maintaining the file directory. The file list tab is always displayed on first entering the file list editor. Click on the Edit File List tab for help on editing the file list.
