Defining and Enabling Topic Free Space Warnings for Topics

Checking this box will tell Moderate to display a warning message box whenever it updates the topic size for this topic if the topic is fuller than the threshold percentage figure you have typed into the edit box.

Appropriate values for the warning threshold depend on how busy a topic is and on how frequently you update your offline copy of its size.

Note that the global Topic size warnings check box must be set as well for warnings to appear.
