Inserting Orphans from the File Directory

Selected from the Insert button menu in the file list dialog.

This dialog shows you all the orphan files in the file directory of your current topic. You can select as many of these as you like; click on the Insert button to add them to your file list. When you have done so, the description editor will be started allowing you to add descriptions to each of the new entries.

File directory entries are flagged by a 'c' in the leftmost column of the file list display.

Click on Cancel if you change your mind about inserting files. The Sort button allows you to resort the list by name, file size or file date to make choosing files easier.
