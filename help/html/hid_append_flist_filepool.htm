Appending File Entries from the Filepool

Selected from the Append button menu in the file list dialog.

This dialog allows you to append file entries that refer to files in the CIX filepool. Type in the name of the file and click on Append to add the entry and invoke the description editor to allow you to add a description to the file.

Filepool entries are flagged by a 'f' in the leftmost column of the file list display.

Note that no validation is performed on the name that you type in; it's up to you to ensure that the file actually exists in the filepool.

Click on Cancel if you change your mind.
