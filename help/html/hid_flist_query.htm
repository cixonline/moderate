Starting the File List Editor on a Topic not Marked as Having a File List

If you attempt to start the file list editor, or switch topics within the editor, to a topic not marked in Ameol as having a file list, the following dialog is displayed:

Select the radio button that is appropriate and click on OK, or click on Cancel if you have made a mistake.

{bmlwd RARROW.BMP}Switch to another topic if you have chosen the wrong topic. Moderate will display a forum/topic selector.

{bmlwd RARROW.BMP}Create file list if the topic has no file list and you wish to create one.

{bmlwd RARROW.BMP}Assume and download if the topic has been erroneously marked as having no file list in Ameol. Moderate will attempt to download the moderator's file list and file directory on your next blink.
