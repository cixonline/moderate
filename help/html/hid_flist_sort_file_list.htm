Sorting File Lists

Selected from the Sort flist button in the file list dialog.

This command allows you to sort the selected lines of your file list. If you have selected more than one region of the list (using control+left mouse button), each selected area will be sorted independently.

Only file entries are sorted. Thus, indented comment lines are kept with their preceding file entry and non-indented comments are kept with their following file entry. Any comment lines at the top of each selected area and any unindented comment lines at the end are not included in the sort.
