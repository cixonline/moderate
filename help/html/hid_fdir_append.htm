Appending Files Into the File List

Selected from the Append button in the file list dialog.

This button appends all the currently selected files to the end of the file list.

Orphan files are appended directly to the end of the edit file list tab display. For files that already have entries in the file list, you will be asked whether you want to append a duplicate entry to the file list:

{bmlwd DUPLICAT.BMP}
