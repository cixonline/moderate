Checking the CIX Forum List

The full CIX forum list is available from Ameol via the CIX Forums Show All Forums menu command. If you haven't already downloaded a copy, Ameol will mark it for download the first time you use this command. It's a good idea to get a fresh copy before creating a new forum as people add at least one new forum to CIX most days.

Look in the sections of the list relevant to the subject of your new forum and don't forget the list of new and unclassified forums at the bottom. It's probably worth searching for relevant keywords using the Find button as well.
