Erasing CIX File Directory Files

Selected from the Action button menu in the file list dialog.

This button allows you to delete the selected files from the file directory. Note that if any have entries in the file list, those entries will be removed.

Depending on how you have the file directory deletion confirmation options set in Settings Moderate..., you may be asked to confirm deletion of all the files you have selected, or for those files that have entries in the file list.
