Working with the File Directory

Selecting the File Directory tab in the file list editor allows you to view and manipulate the file directory for your topic. You can insert or append files into the file list from here, and rename, delete or download them regardless of whether they have entries in the file list or not.

If you are in any doubt whether a file someone has uploaded is legal (or virus-free if it is a program), you can download it from here and check it prior to adding it to the file list.

You can sort the file directory display by file name, size or date to make it easier to find the files you are interested in and you can use the radio buttons to control whether only orphan files, file with entries in the flist or all files in the directory are displayed.
