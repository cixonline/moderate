Selecting a Topic for the File List Editor

If you do not have a message window open on a topic in a forum you moderate when you select Topic Edit File List, Moderate will display the the following topic selection dialog:

Choose the forum and topic you want to work on and click on OK, or click on Cancel if you have changed your mind.

If you choose a topic not marked in Ameol as having a file list, Moderate will display another dialog asking you what you want to do.
