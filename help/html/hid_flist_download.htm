Downloading Files from a File List Using Moderate

Selected from the Action button menu in the file list dialog.

The Download command allows you to download files directly from the file list editor. The files will be downloaded to Moderate's download directory (set in Settings Moderate) - this defaults to Ameol's download directory when Moderate is first installed.

A separate default directory is provided so that files download as part of your moderating duties can be kept separate from those you download in your general CIX usage.
