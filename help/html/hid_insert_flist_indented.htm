Inserting Indented Comments into a File List

Selected from the Insert button menu in the file list dialog.

Selecting Indented from the Insert menu inserts an indented comment line into your file list at the current cursor position. This line will be indented to start at the same point as that of file descriptions.

The description editor is then started allowing you to type in whatever text you like; this text can extend over more than one line, the indented comment in your file list being expanded as required when you leave the description editor.

Note that this command is not needed to enter file descriptions that extend over more than a single line; Moderate will add indented lines automatically if you type in a file description that is longer than one line.
