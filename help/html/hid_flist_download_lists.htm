Refreshing Your Offline Copies of the File Lists

Selected from the Refresh button in the file list dialog.

Select the checkboxes for the files you wish to update from CIX.

If either the moderator's file list or the file directory are missing, this dialog will be displayed when you first select Topic Edit File List with the appropriate entries checked but disabled. You will be able to use the file list editor after your next blink downloads the missing files to your computer.
