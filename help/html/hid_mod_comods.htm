Coordinating Moderating Duties Between Co-moderators

If your forum is busy or has many topics covering different subjects, you may want to share the workload of moderating it.

You can add (or remove) co-moderators with Ameol's CIX Forums Participants... command; it's worth noting here that all co-moderators of a forum have exactly the same privileges, if you add someone as co-moderator, they can remove you... :-)

How you organise moderating responsibilities is up to you, but in general, it's better to either have a rota or to allocate responsibility for particular topics. If you're all trying to moderate the same topics at the same time, things either get left because you all assume someone else is dealing with it, or a question from a participant can end up with half a dozen identical replies.

The most important consideration is the updating of file lists; conflicts here can result in serious problems. See Coordinating File List Updates Between Co-moderators for how to get round this.
