Adding a File List to an Existing Topic

If you want to add a file list to an existing topic that was created without one, first of all make sure that Ameol hasn't the topic incorrectly marked as having a file list. If it has, remove the checkmark from the Topic can have files flag in the Settings dialog for the topic, accessible from the Ameol forum list with your topic highlighted.

You then need to start the file list editor on the topic using the CIX Forums Edit File List command. You will be presented with the following dialog:

Make sure that the "Create file list" radio button is checked as shown and click on the OK button. This will take you into the file list editor with a blank file list. Do not insert any files at this stage, just put in any descriptive comment lines you wish and click on the Upload flist button.

When you next blink, the file list will be created on CIX. Thereafter you can edit it in the normal way.
