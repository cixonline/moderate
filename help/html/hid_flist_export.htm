Exporting File Entries to the Filepool

Selected from the Action button menu in the file list dialog.

This dialog is displayed for each file selected that either refers to a local file, or refers to a filepool file with a local fdir file available with the same name.

Notes

{bmlwd RARROW.BMP}When exporting or transforming from local to filepool, no check is made whether the file is already in the filepool.

{bmlwd RARROW.BMP}If a local file entry in the file list is converted to a filepool entry the file in the file directory is not erased.
