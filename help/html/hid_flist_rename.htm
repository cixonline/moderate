Renaming a CIX File Directory File

Selected from the Action button menu in the file list dialog.

This dialog is displayed for each file you have selected for renaming from the file directory dialog.

Type in the new name the file should have on CIX. If the file has an entry in the file list, that entry will be renamed automatically.

Note that the Cancel all button cancels all remaining rename operations if you have selected more than one file from the file directory.
