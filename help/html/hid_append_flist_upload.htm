Appending Entries by Uploading from Your Computer

Selected from the Append button menu in the file list dialog.

This dialog is a standard file selector. The initial directory is set to whatever you have your upload directory set to in Moderate's settings (this in turn defaults to your Ameol upload directory). The file selector allows multiple selections; when you have selected the files you want, click on OK and the description editor will be started allowing you to add descriptions to each of the new entries.

The files you select will be uploaded to the topic on your next blink.

Until you leave the file list editor (or change topics) your new entries will be flagged by a 'u' in the leftmost column of the file list display.

Click on Cancel if you change your mind about appending files.
