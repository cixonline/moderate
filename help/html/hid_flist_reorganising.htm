Formatting a File List

You can insert descriptive comment lines and blank lines anywhere in your file list to clarify its layout. If the file list is more than a screen or so long, it may be worth subdividing it into sections to make it easier for your participants to find files.

You can move lines, or marked blocks of lines, up and down your file list using the Move up and Move down buttons.

You can sort file entries in the file list by highlighting the block (or blocks) of files you want to sort and selecting Sort flist. Moderate will sort each contiguous block of marked lines independantly. Indented continuations of file descriptions are kept with the relevant entry, embedded comment lines are kept with the following file entry (any comment lines at the top or bottom of the marked block are just left where they are).
