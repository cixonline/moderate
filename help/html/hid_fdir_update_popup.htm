Updating Files in the File Directory

on the file list dialog

{bmlwd UPDFDIR.SHG}Update tells Moderate that you want to replace one or more files in the file directory. You use a file you have already uploaded to your mail directory or a file on your local hard disk to replace the existing file on CIX.

When you click on the Update button, this menu pops up. Click on the menu options for more information on each update option.
