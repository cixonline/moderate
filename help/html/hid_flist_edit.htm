Editing Entries in a File List

Selected from the Edit button in the file list dialog or by double-clicking on entries in the file list.

Selecting Edit starts the Moderate description editor. If you have more than one entry in the file list selected, the Next and Previous buttons are enabled in the editor allowing you to step through the files you have selected. Click on the Done all button when you have finished editing all the descriptions.

Note that indented description lines following a file description and sequences of comment lines are treated as single entries for editing purposes; Moderate takes care of formatting these and adjusting the number of lines used automatically for you.

The appearance of description editor dialog varies according to whether you are editing a file entry, a filepool file entry, a comment, a tabbed comment or a private memo.

The Undo button cancels changes you have made to the current entry you are editing; you cannot undo changes after switching away from an entry using Next or Previous.

In all cases the Today's date button inserts today's date into the description in the format defined in Settings Moderate. When editing file entries, the File date button is enabled and inserts the modification date of the file whose description is being edited.
