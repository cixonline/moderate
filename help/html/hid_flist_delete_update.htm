Removing, Hiding and Updating Files

You can delete files from a file list by highlighting their entries and selecting Delete from the Action button popup menu. This will remove their entries from the file list and, depending on how you have Moderate configured, will also delete them from the file directory.

Entries can be temporarily hidden (if there is some doubt about whether a file is legal to post on CIX, for example) using Hold from the Action button popup menu. Held entries are greyed out in the file list editor display, and are invisible in the users' file list. To release a held file entry, just select Hold again.

You can update a file with a new version with the same name from your binmail directory using Update Mail dir or from your local hard disc using Update Upload. In the latter case, the file on your hard disc needn't have the same name as that of the flist file being updated; Moderate will rename it for you after uploading it to CIX. If you update files, don't forget to edit the file list description if required, and post a notification message so your participants know that the file has changed.
