Appending Private Memos to a File List

Selected from the Append button menu in the file list dialog.

Selecting Memo from the Append menu adds a private note to the end of your file list. This note is only visible when you are editing the file list, it will not appear in the users' file list.

The description editor is then started allowing you to type in whatever text you like; this text can extend over more than one line, the private memo in your file list being expanded as required when you leave the description editor.
