Inserting Files Into the File List

Selected from the Insert button in the file list dialog.

This button inserts all the currently selected files into the file list at the current caret position in the file list display.

Orphan files are inserted directly at the cursor position in the edit file list tab display. For files that already have entries in the file list, you will be asked whether you want to add a duplicate entry to the file list:
