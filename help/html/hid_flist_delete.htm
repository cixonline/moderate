Deleting Entries from a File List

Selected from the Action button menu in the file list dialog.

Selected file list entries that do not refer to files are deleted without confirmation. If only one file is selected, you will be asked for confirmation for that file; if more than one file is selected, the following dialog will be displayed:

This allows you to control confirmation of deletion of each file and whether the associated entries in the file directory are erased as well.

The default settings for this dialog can be changed using Settings Moderate.
