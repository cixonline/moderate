Inserting Private Memos into a File List

Selected from the Insert button menu in the file list dialog.

Selecting Memo from the Insert menu inserts a private note into your file list at the current cursor position. This note is only visible when you are editing the file list, it will not appear in the users' file list.

The description editor is then started allowing you to type in whatever text you like; this text can extend over more than one line, the private memo in your file list being expanded as required when you leave the description editor.
