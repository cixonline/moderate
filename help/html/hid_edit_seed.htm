Edit Seed Message

This message editor allows you to edit the seed message that Moderate creates when you add a topic to a forum. By default the title of the message is the description that you write for the topic, and the body contains:

Welcome to <forum>/<topic>
This topic has a file list
This topic is read-only

The latter two lines being added you have set the relevant checkboxes in the topic creation dialog.

Note that the Undo button undoes all changes you have made in the editor.
