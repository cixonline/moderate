Hiding and Restoring File List Entries

Selected from the Action button menu in the file list dialog.

The Hold command allows selected entries in a file list to be made temporarily invisible to users. The command acts as a toggle, allowing entries to be switched between visible and invisible at will.

Hidden entries are greyed out in the file list display; hidden comment lines are further identified by an 'h' in the leftmost column in order to distinguish them from private memo lines.

Hidden entries are tagged with a special header in the moderator's file list.
