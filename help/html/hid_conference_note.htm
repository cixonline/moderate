The Forum Note

When you look at the details of a forum (either using the Ameol ShowConf addon or by using the CIX "show <conf name>" command whilst online), you sometimes get extra text after the one-line forum description and the list of topics, telling you about the forum in more detail.

This is called the Forum note. As Moderator you can create and edit this text by using the "Conf Note" button in either the CIX Forums Create Forum or the CIX Forums Edit Topics dialogs.

This is a good place to put any special information about your forum that people should know before they join.
