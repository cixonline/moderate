Inserting New Entries in a File List

on the file list dialog

Insert tells Moderate that you want to insert one or more new file list entries above the current cursor position in the file list. (You can also add new entries to the end of the file list - see Appending New Entries to a File List.)

When you click on the Insert button, this menu pops up. Click on the various menu options for more information on each insertion option.
