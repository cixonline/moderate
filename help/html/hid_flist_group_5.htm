The File Directory Display

on the main file list dialog

The File Directory tab display shows the name, size and last modified date for every file in the file directory that is selected for display. You may choose to display orphan files, files that have an entry in the file list or all files using the radio buttons at the bottom right of the file directory display dialog.

Orphan files are marked as such by an 'o' at the far left of the file directory display.

You can set the default selection of files to be displayed when you first view the file directory in Settings Moderate....
