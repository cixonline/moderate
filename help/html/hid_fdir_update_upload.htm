Updating Files by Uploading from Your Computer

Selected from the Update button menu in the file list dialog.

This dialog is a standard file selector. The initial directory is set to whatever you have your upload directory set to in Moderate's settings (this in turn defaults to your Ameol upload directory). The file selector is invoked for each filename you have selected in the file list.

When you have selected the file you want to use to update the file on CIX (it needn't have the same name as the CIX file), click on OK. The old file on CIX will be deleted and the file you have selected will be uploaded and, if necessary, renamed to match the old file on CIX.

Click on Cancel if you change your mind about updating any particular file.
