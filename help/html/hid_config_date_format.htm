Date Format for File List Descriptions

You can set the format Moderate uses for dates you insert into file descriptions. This format is the same as is used by the standard C library routine strftime().

You can put in any plain text plus any of the following formatting sequences:

%%character %
%aabbreviated weekday name
%Afull weekday name
%babbreviated month name
%Bfull month name
%cdate and time, default format
%d2-digit day of month (00-31)
%H2-digit hour (00-23)
%I2-digit hour (01-12)
%j3-digit day of year (001-366)
%m2-digit month as a decimal number (1-12)
%M2-digit minute (00-59)
%pAM or PM
%S2-digit second (00-59)
%U2-digit week number (Sunday first day of the week) (00-53)
%wweekday where 0 is Sunday (0-6)
%W2-digit week number (Monday first day of the week) (00-53)
%xdate, default format
%Xtime, default format
%y2-digit year without century (00-99)
%Yyear with century
%Ztime zone name, or no characters if no time zone
