Completing the Editing of a File List

Selected from the OK button in the file list dialog.

This button accepts any changes you have made to the file list you have been editing and returns you to Ameol proper.

If you have made any changes to the file list and haven't asked for it to be uploaded to and updated on CIX, Moderate will either upload and update automatically or ask you if you want the file list to be updated on CIX.

Moderate will update the file list on CIX automatically if you have made any changes to the file directory that would invalidate the old file list (either renaming or deleting files).
