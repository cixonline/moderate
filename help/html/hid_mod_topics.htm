Looking After Topics in Your Forum

Topic size

There is a limit to the size of each topic on CIX; at present this is 2 megabytes, we are intending to increase or abolish this limit at some time in the near future. In the mean time, the Moderate CIX Forums Edit Topics dialog displays the size of each topic in forums that you moderate.

Clearly, this information needs to be periodically updated from CIX. You can configure Moderate to update the topic size for each topic (or for all topics in a forum) either manually only, every 'n' days or every time you blink.

You can also configure Moderate to warn you when a topic gets over a configurable percentage full.

Pruning forums

Withdrawn messages are not physically deleted by CIX so if you are running short of space in a topic, you may wish to physically remove the withdrawn messages in that topic. To do this select the Prune button from the CIX Forums Edit Topics dialog. Pruning a topic on CIX can take a long time and the topic is temporarily made read-only while the topic is reorganised, so you should consider doing this at a non busy time, if possible.

Adding Topics

If there is a need for a new topic in your forum or if an existing topic is full, you can add new topics from the CIX Forums Edit Topics dialog.

By convention, replacements for full topics have the same name prefixed with a digit. For example, queries would be supplanted by 2queries, which in turn would be replaced by 3queries, and so on.

All your participants should be automatically joined to any new topic you create, but occasionally this can fail to happen. Thus it is best to announce the existance of a new topic elsewhere in your forum to make sure that everyone knows about it.

Removing Topics

It is possible to delete topics from a forum, from the CIX Forums Edit Topics dialog. This is usually only done if the topic is either full or obsolete and contains ephemeral data that it is unlikely that anyone will want to refer back to; unless you have asked cixtech to archive your topic first, it is impossible to recover a deleted topic.
