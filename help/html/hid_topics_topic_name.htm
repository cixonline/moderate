Topic Name

You can use this field to select an existing topic to deal with. Either use the cursor up and down keys to step through the list or type in the first letters of the topic name. If you type in a prefix common to more than one topic, you can step through those topics using the space bar.

If you are creating a new topic (after clicking the New Topic button), type in the name of your new topic here. When you have typed in the topic description and set the file list and read only flags, click on the Add button to register the topic for creation.

Topic names on CIX can be up to 14 characters long and can contain lower case letters, digits, plus and minus signs, underscores and full stops. Certain names are not permitted by CIX, news and mail for example. The name you type in will be checked against these as soon as you move to another field in the dialog.
