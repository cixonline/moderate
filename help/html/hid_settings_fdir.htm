Moderate command (Settings Menu)

File Directory Settings

This property sheet allows you to set various configuration options in Moderate to suit the way you want to work. For Ameol, these options are saved into your current Ameol initialisation file in a section called [Moderate], for Ameol32 they are saved into the registry. (They can be edited by hand, but it is strongly recommended that you use this dialog to make any changes.)

Click on the controls in the property sheet for information on what they do.

Click on the tabs for help on general and file list settings.
