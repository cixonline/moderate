Appending Entries Notified to You by Mail

Selected from the Append button menu in the file list dialog.

This dialog allows you to append file entries for files whose existance you have been informed of by mail without having to update your offline copy of the file directory. Type in the name of the file and click on Append to add the entry and invoke the description editor to allow you to add a description to the file.

Until you leave the file list editor (or change topics) your new entry will be flagged by a 'n' in the leftmost column of the file list display.

Note that no validation is performed on the name that you type in.

Click on Cancel if you change your mind.
