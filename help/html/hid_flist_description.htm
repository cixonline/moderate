File Lists on CIX

The four file lists on CIX relevant to file list maintenance are:

The File Directory

Commonly known as the fdir. This is the actual directory on CIX where files uploaded to a topic are stored; CIX creates the list dynamically every time Moderate requests it to be downloaded. When you are editing the file list in Moderate, you can choose files from here to add to the file list visible to users.

The fdir is stored on your computer as <topic root>.fld in the flist subdirectory of you Ameol data directory.

The Filepool

This is a pool of files held on CIX that is accessible from any topic file list. Files of general interest can be exported here from a topic file list whereupon they can be included in the file list of any other topic.

The CIX forum filepool maps directly onto the filepool; if a user uploads a file to the 2newfiles topic of this forum, it can be accessed by moderators of any forum on CIX.

Moderate doesn't keep a copy of the filepool fdir on your computer; if you want to add filepool files to your topic file list, you have to type the file's name in yourself.

The Moderator's File List

This is the file list you actually edit in Moderate. CIX uses it to create the users' file list whenever a copy is requested; the file references in the moderator's file list are matched against the files in the fdir or filepool and the list as displayed by Topic File List in Ameol is created.

The moderator's file list is stored on your computer as <topic root>.flm in the flist subdirectory of you Ameol data directory; a single backup copy is also kept by Moderate as <topic root>.flb.

The Users' File List

This is the file list that is displayed by Ameol's Topic File List command, and the only list from which your forum participants can download files. CIX generates it dynamically from the moderator's file list, the fdir and the filepool whenever Ameol requests a copy.

The users' file list is stored on your computer as <topic root>.fls in the flist subdirectory of your Ameol data directory.
