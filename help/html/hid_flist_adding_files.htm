Adding Files to a File List

Files can be added to a file list from the file directory, from the filepool, from your mail directory and by upload from your computer. You can also either insert new files wherever the caret is in the file list display or append them to the end.

If one of your participants uploads a file, it will end up in the file directory. Hopefully they will also send you a mail message telling you what they've uploaded and where. You can add these files using Insert (or Append)Fdir providing you've updated your offline copy of the fdir to reflect the new addition(s). If you have received a notification, you can use Insert (or Append)Notification and just type in the filename(s) instead.

You can add files in your binmail directory (either files you have uploaded or files binmailed to you) using Insert (or Append)Mail dir; you may need to update your offline copy of your binmail directory first.

You can add files directly from your hard disk using Insert (or Append)Upload; the file(s) will be uploaded and added to the file list on your next blink.

In all cases, you will get the file list entry description editor dialog displayed so that you can add a description to each file you add to the file list. The description can be as long as you like; Moderate will add indented continuation lines for you as required.

You can create notification messages for your new entries by selecting them in the file list editor list box and selecting Notify.
