Importing File Descriptions

Selected from the Edit button in the file list dialog.

Moderate allows file descriptions to be imported wholesale from a text file. This can be used, for example, to rebuild a moderator's file list from the users' file list if it has accidentally been corrupted on CIX.

Certain assumptions are made by Moderate when processing an import file.

This is a standard file selector, but note the extra Description field number selector. This must be set to the column position (counting from one) where the file description starts.
