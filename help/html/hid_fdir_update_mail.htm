Updating Files from Your Binmail Directory

Selected from the Update button menu in the file list dialog.

Selecting Mail dir from the Update menu with one or more files selected in the file list will tell Moderate to search for files with the same names in your binmail directory. If a file is found with the same name as one you have selected, the old file in the CIX topic will be deleted and the new one exported from your mail directory to the topic.

If you don't currently have a copy of your binmail directory available offline, Moderate will offer to download it on your next blink.
